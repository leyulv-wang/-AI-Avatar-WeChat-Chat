# 页面设计：三栏聊天工作台（桌面端优先）

## 全局规范（Global Styles）
- 设计风格：接近微信桌面端的克制风格（高密度信息 + 清晰分隔）
- 背景色：#F5F5F5（页面背景）；面板背景 #FFFFFF
- 分割线：#E5E7EB；阴影：轻量（如 `shadow-sm`）
- 字体：系统字体栈；正文 14px；辅助信息 12px；标题 16px
- 主色（强调/可点击）：#07C160（可用于“发送/在线状态点”）
- 按钮：
  - Primary：绿底白字；hover 加深
  - Ghost：透明底 + 边框；hover 浅灰底
- 链接/可点击文本：hover 下划线或颜色加深
- 动效：仅使用轻量过渡（150–200ms），避免聊天滚动抖动

## 响应式策略（Desktop-first）
- ≥ 1200px：三栏固定展示（左/中/右）
- 992–1199px：右栏可折叠为抽屉（默认收起，按钮展开）
- ≤ 991px：优先保证中栏；左栏变为抽屉；右栏与候选回复入口放在中栏上方标签区

## 页面：聊天工作台（/）

### Meta Information
- title：三栏聊天工作台
- description：联系人列表、仿微信聊天记录与候选回复的高效聊天界面
- Open Graph：
  - og:title 同 title
  - og:description 同 description

### Layout
- 主布局：CSS Grid（推荐）
  - `grid-template-columns: 280px minmax(520px, 1fr) 320px;`
  - 列间距：0（用分割线）；面板内边距：12–16px
- 纵向：三栏各自独立滚动区域，避免整页滚动影响聊天体验
- 分层：顶部状态栏固定（sticky），各栏内容区滚动

### Page Structure
1) 顶部状态栏（通栏）
2) 主体三栏：左栏联系人列表 / 中栏聊天 / 右栏候选回复

### Sections & Components

#### 1) 顶部状态栏（Top Bar）
- 左侧：应用名（可简写） + 连接状态（圆点 + 文案：在线/重连中/离线）
- 中间：当前会话标题（未选中会话时显示“请选择联系人”）
- 右侧：刷新按钮（用于手动触发数据重新拉取/重连）
- 状态：
  - 离线：顶部显示浅红提示条（不遮挡内容）
  - 重连中：显示 spinner + 文案

#### 2) 左栏：联系人/会话列表（Contacts Panel）
- 顶部：搜索框（placeholder：搜索联系人/会话）
- 列表项（List Item）：
  - 头像（32–40px）
  - 两行文本：昵称 + 最新消息摘要
  - 右侧：未读 badge（0 不显示）
- 交互：
  - 点击列表项：高亮选中；触发加载聊天记录与订阅切换
  - 空态：无联系人/会话时显示说明文案
  - 加载态：骨架屏（3–6 行）

#### 3) 中栏：聊天记录 + 输入区（Chat Panel）
- 聊天记录区（可滚动）：
  - 时间分隔：居中灰字（如“10:21”或“昨天”）
  - 消息气泡：
    - 对方：左对齐，浅灰气泡
    - 自己：右对齐，绿色或浅绿色气泡
  - 发送状态：
    - sending：气泡旁小 spinner
    - failed：红色小提示“发送失败，点击重试”（如你现有后端支持重试）
  - 新消息提示：当用户不在底部时，底部浮层按钮“有新消息”点击滚到底
- 输入区（固定在中栏底部）：
  - 多行文本框（1–5 行自适应）
  - 右侧发送按钮；Enter 发送，Shift+Enter 换行
  - 当未选择会话时：输入区禁用并提示“请先选择联系人”

#### 4) 右栏：候选回复（Candidate Replies Panel）
- 顶部：标题“候选回复” + 刷新/加载状态（小字）
- 列表：卡片（每条候选 1 张卡）
  - 内容：候选回复文本（最多 4–6 行，超出折叠）
  - 可选 meta：标签/理由（若后端提供）
  - 操作：
    - 主按钮：填入输入框
    - 次按钮：直接发送
- 状态：
  - 加载态：骨架卡片
  - 空态：显示“暂无候选回复（等待新消息或手动刷新）”

### Interaction & Real-time Guidelines
- 会话切换时：
  - 中栏先显示加载态，再渲染历史；实时订阅切换需先取消旧会话订阅
- 实时到达消息：
  - 若滚动在底部：自动滚到底
  - 否则：不抢滚动，仅显示“有新消息”提示
- 候选回复刷新：
  - 当新消息到达或会话切换时触发刷新；若后端推送则以推送为准
- 错误提示：
  - 仅在对应区域展示（联系人区/聊天区/候选区），避免全局弹窗打断聊天
